# 📘 Exercícios - USJT

Este repositório reúne exercícios propostos para serem feitos em casa, com o objetivo de praticar, testar conhecimentos e desenvolver o raciocínio lógico ao longo do curso.

## 💡 Sobre

Os exercícios aqui presentes fazem parte das atividades da faculdade e servem como complemento às aulas, ajudando no aprendizado contínuo e na fixação dos conteúdos.

## 📌 Observações

Este material é destinado apenas para estudo e prática individual.
